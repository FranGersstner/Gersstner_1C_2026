void BcdToGpio ( uint8_t digit,
gpioConf_t * vec )

Mapea un dígito BCD a un arreglo de pines GPIO.

Realiza un desplazamiento de bits (bit shift) lógico sobre el dígito BCD para evaluar el estado de cada uno de sus 4 bits constitutivos. Dependiendo de si el bit es '1' o '0', enciende (GPIOOn) o apaga (GPIOOff) el pin de datos correspondiente.

  • Parámetros
    [in]digitDígito BCD (valor entre 0 y 9) a representar en hardware.
    [in]vecPuntero al arreglo de configuración de pines GPIO (debe contener NRO_BITS_BCD elementos).

void Convertir_num_y_mostrar ( uint32_t number,
uint8_t digits,
gpioConf_t * gpioVec,
gpioConf_t * selVec )

Convierte un número a BCD y lo envía secuencialmente al display.

La función coordina el proceso de visualización en dos pasos. Primero, llama a convertToBcdArray para obtener los dígitos. Segundo, itera sobre cada dígito llamando a BcdToGpio para establecer los valores lógicos en los pines de datos, y genera un pulso activando y desactivando el pin de selección correspondiente.

  • Nota
    Debido a la ausencia de retardos (delays) entre el encendido y el apagado del pin de selección, esta función ejecutará un barrido instantáneo que no será visible para el ojo humano en un entorno físico.
  • Parámetros
    [in]numberNúmero entero que se desea mostrar en el display.
    [in]digitsCantidad de dígitos que tiene el display.
    [in]gpioVecPuntero al arreglo de configuración de pines de datos (BCD).
    [in]selVecPuntero al arreglo de configuración de pines de selección (Dígitos).
  • Devuelve
    Ninguno (void).

void convertToBcdArray ( uint32_t data,
uint8_t digits,
uint8_t * bcd_number )

Convierte un número entero a un arreglo de dígitos BCD.

Toma un valor entero y extrae cada uno de sus dígitos utilizando operaciones sucesivas de módulo y división por 10. Los dígitos se almacenan en el arreglo provisto, ordenados de forma que el dígito más significativo quede en la primera posición (índice 0).

  • Parámetros
    [in]dataNúmero entero (de hasta 32 bits) que se desea convertir.
    [in]digitsCantidad de dígitos que se extraerán del número.
    [out]bcd_numberPuntero al arreglo donde se guardarán los dígitos BCD extraídos.
